# avalon-todomvc

> [TodoMVC](http://todomvc.com/) done with [avalon](https://github.com/ryanmorr/avalon)

## Install

```sh
git clone git@github.com:ryanmorr/avalon-todomvc.git
cd avalon-todomvc
npm install
npm start
```

Then open `http://localhost:8080` in your browser.
